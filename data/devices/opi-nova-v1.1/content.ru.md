| Категория | Технические характеристики |
| ---- | ---- |
| Производитель | Xunlong Software (OrangePi) |

| Процессор | 2K3000 с тактовой частотой 2,2 ГГц |
| Память | DDR4 со скоростью 3200 МТ/с (встроенная память) |

| Интерфейсы (видео) | 1 × HDMI, 1 × MIPI (30-контактный), 1 × eDP |
| Интерфейсы (прочее) | 1 разъем m.2 Key E (PCIe + USB), 1 разъем 3,5 мм для наушников/микрофона, 1 порт UART процессора, 1 порт UART системного модуля, 40-контактный разъем GPIO |
