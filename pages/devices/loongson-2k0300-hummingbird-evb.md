龙芯 2K0300 蜂鸟板是由龙芯中科推出的，基于 2K0300 平台设计的开发板。

## 主要规格

| 类别 | 规格 |
|------|------|
| 厂商 | 龙芯中科 |
| 板型 | 93×63mm |
| 处理器 | 2K0300 @ 1.0GHz |
| 内存 | 512MiB DDR4（板载） |
| 接口 (USB)  | 2 × USB 2.0、1 × USB-C† |
| 接口（存储）| 8GB eMMC（板载） |
| 接口（网络） | 1 × GbE |
| 接口（视频） | 1 × LCD 输出 |
| 接口（其他） | 2 路 SPI、4 路 PWM、4 路 CAN-FD、40-pin GPIO |

†：USB-C 接口仅可用于供电和连接串口。
