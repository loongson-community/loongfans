## Обновления прошивки/системного интерфейса

- Удален код обратной совместимости для устаревших сред (ABI 1.0; точнее, «Спецификация унифицированной системной архитектуры процессоров Loongson» версии 3.1), при этом все платформы переведены на спецификацию версии 4.0 ([Ссылка](https://github.com/tianocore/edk2/)).
- Перешли с компилятора GCC 8.3 на GCC 12/13/14, полностью перейдя на инструментарий сообщества EDK2.
- Из интерфейса настройки были удалены устаревшие параметры режима загрузки; теперь полностью используется соотношение адресов виртуальной памяти (VA) и физической памяти (PA) 1:1.
- Переход с устаревшей версии UDK2018 на EDK2025; теперь большинство инфраструктурных API LoongArch соответствуют реализациям EDK в основном репозитории.

## Обновления модулей

- Добавлена поддержка платформ на базе одно- или многопроцессорных платформ 3B6000, 3C6000/{S,D,Q} и 2K3000/3B6000M.
- Обновлен код для различных программных и аппаратных модулей, таких как PHY, MRC, SMC, IPMI, MultiArch, eMMC и т. д.

## Новые функции

Общие обновления функций:

- Добавлена функция MultiArchUefiPkgs для поддержки эмуляции x86 Option ROM/EFI и фильтрации периферийных устройств сторонних производителей (может включаться/выключаться в настройках прошивки).
- Добавлена возможность настройки частот памяти — при необходимости можно снизить тактовую частоту памяти для обеспечения лучшей совместимости (доступные для настройки частоты могут различаться в зависимости от модели; в интерфейсе настройки прошивки предусмотрены опции для восстановления настроек; также можно снять батарейку CMOS для сброса настроек).
- Добавлены настройки для изменения разрешения интерфейса и консоли.
- Добавлена опция DSM#5 для управления стратегией распределения ресурсов PCIe в ядре Linux.
- Добавлена поддержка периферийных устройств китайского производства, в том числе RAID-контроллеров HRDT и сетевых карт 25/40GbE компании BZWX.
- Добавлены настройки для работы с несколькими графическими процессорами, что позволяет одновременно отображать изображение с помощью разных графических процессоров в среде UEFI.
- Добавлены функции TPM и Secure Boot с поддержкой таких алгоритмов, как SHA-384, SHA-512 и SM3 (требуется аппаратная поддержка TPM).
- Разделены окна с информацией о PCIe, жестких дисках и USB; добавлено отображение подробной информации о жестких дисках.
- Обновлена версия SRAT до 6.5 с добавлением поддержки расширения x2APIC.
- Синхронизирована логика кода, касающаяся пробуждения многоядерных процессоров LoongArch, обработки исключений, MMU и т. д., с исходным кодом EDK2.

Особенности конкретной платформы:

- Добавлена поддержка инициализации устройств PCIe 4.0 для платформ Loongson Coherent Link (LCL) — серий 3B6000 и 3C6000/{S,D,Q}.
- Добавлена поддержка eMMC 2K3000.
- Добавлена поддержка мониторинга тепловых зон ACPI и HWMon для 3C6000.
- Добавлена поддержка ACPI EDAC для серверных платформ, что позволяет запрашивать состояние ECC из Linux.
- Улучшенное ограничение мощности на базе SMC, функция «Turbo Boost» и мониторинг/управление вентиляторами для отдельных моделей.
- Поддержка отчетности по описанию ресурсов модуля безопасности (SE) через таблицы DSDT на отдельных моделях.

## Исправления и оптимизации

Общие исправления:

- Устранены потенциальные сбои системы при выборе «Все видео» в качестве основного дисплея на серверных платформах с одним процессорным разъемом, на которых установлены как дискретные видеокарты, так и видеокарты BMC.
- Исправлено некорректное отображение версии прошивки и времени сборки на экране загрузки.
- Оптимизирована поддержка таблицы SMBIOS, в первую очередь исправлены логические ошибки, связанные с DMI Type 7.
- Устранены неожиданные зависания при операциях записи в RTC LP.
- Исправлены известные проблемы с таблицами ACPI.
- Исправлена некорректная последовательность переключений контроллера портов SATA.
- Исправлено несколько ошибок в китайском переводе.
- Исправлено периодическое исчезновение логотипа во время загрузки.

Исправления и настройки для конкретных платформ:

- Исправлена проблема, из-за которой на некоторых моделях могла не отображаться информация о сетевых картах GMAC.
- Частота работы чипа моста 7A2000 на серверных платформах снижена до 750 МГц.
- Оптимизирована совместимость протоколов команд BMC для моделей AST2500 и LS2K0500; устранены проблемы с отображением данных IPMI и ошибки при выполнении определённых команд.
- Исправлена проблема с рассинхронизацией информации SMBIOS в веб-интерфейсе BMC при использовании BMC LS2K0500 с версиями прошивки выше v2.2.2.
- Исправлены проблемы, связанные с обратной полярностью разъемов SATA и некорректными настройками PCIe LnkCap2 на некоторых платформах.
- Исправлена ошибка 2K0500 в выводе BMC Serial-over-LAN (SOL).
- Исправлены таблицы PPTT для нескольких чипов.
- Отключена функция PTW для предотвращения повреждения данных, вызванного проблемами с таблицей страниц ядра Linux (будет включена вновь, как только исправления для ядра станут широко доступны).
- Удалены описания ресурсов ACPI GPIO для некоторых платформ во избежание проблем с совместимостью (будут восстановлены, как только поддержка в ОС станет широко доступной).

Функциональные оптимизации:

- Улучшены различные интерфейсы настройки прошивки.
- Улучшенный интерфейс обнаружения и настройки устройств BMC, включающий расширения данных FRU, элементы управления интерфейсом USB-портов и многое другое.

Вышеуказанная информация первоначально предоставлена компанией Loongson Technology [Примечания к выпуску прошивки](https://github.com/loongson/Firmware/blob/main/docs/changelist_V5.0.0343-stable2511.md); переработан и в нескольких местах перефразирован для большей понятности.
