The CTCISZ 3B6000M mini PC is a mini PC similar to Intel NUC specifications, equipped with a 3B6000M processor and providing dual HDMI outputs and dual gigabit (GbE) Ethernet ports.

## Main Specifications

| Category | Specification |
|------|------|
| Manufacturer | CTCISZ |
| Processor | 3B6000M @ 2.5GHz |
| Memory | DDR4 @ 3200MT/s (1 × SODIMM) |
| Interface (USB)  | 3 × USB 3.0, 3 × USB 2.0, 2 × USB-C† |
| Interface (Storage) | 1 × NVMe (PCIe 3.0 x4) |
| Interface (Network) | 2 × GbE |
| Interface (Video) | 1 × DisplayPort, 1 × HDMI |
| Interface (Other) | 1 × m.2 Key E interface (PCIe + USB), 1 × 3.5mm headphone/microphone jack |

†: One of the USB-C interfaces can only be used to power the board and has no data functionality.
