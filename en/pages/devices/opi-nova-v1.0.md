The OrangePi Nova v1.0 is a Nano-ITX (120×120mm) motherboard based on Loongson 2K3000.

## Key Specifications

| Category | Specification |
|------|------|
| Manufacturer | Xunlong Software (OrangePi) |
| Form Factor | Nano-ITX (120×120mm) |
| Processor | 2K3000 @ 2.2GHz |
| Memory | DDR4 @ 3200MT/s (onboard) |
| Interfaces (USB)  | 2 × USB 3.0, 2 × USB 2.0, 2 × USB-C† |
| Interfaces (Storage) | 1 × NVMe (PCIe 3.0 x4), 1 × microSD card slot, 32GB eMMC (onboard) |
| Interfaces (Network) | 1 × 2.5GbE (Realtek RTL8125), 1 × GbE |
| Interfaces (Video) | 1 × HDMI, 1 × MIPI (30-pin), 1 × eDP |
| Interfaces (Other) | 1 × m.2 Key E interface (PCIe + USB), 1 × 3.5mm headphone/microphone jack, 1 × CPU UART, 1 × SE UART, 40-pin GPIO |

†: One USB-C port is for power delivery only, without data transfer capability.
