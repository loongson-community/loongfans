The Loongson 2K0300 Hummingbird board is a development board launched by Loongson Technology, designed based on the 2K0300 platform.

## Main Specifications

| Category | Specification |
|------|------|
| Manufacturer | Loongson Technology |
| Form Factor | 93×63mm |
| Processor | 2K0300 @ 1.0GHz |
| Memory | 512MiB DDR4 (onboard) |
| Interface (USB)  | 2 × USB 2.0, 1 × USB-C† |
| Interface (Storage) | 8GB eMMC (onboard) |
| Interface (Network) | 1 × GbE |
| Interface (Video) | 1 × LCD output |
| Interface (Other) | 2 × SPI, 4 × PWM, 4 × CAN-FD, 40-pin GPIO |

†: The USB-C interface can only be used to connect to serial port.
